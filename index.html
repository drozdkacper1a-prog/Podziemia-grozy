Hosting statyczny

Wrzuć game.html na GitHub Pages, Netlify, Vercel lub własny serwer. Ustaw src iframe na URL pliku.

📁 game.html → twoja-domena.pl/game.html

WordPress / CMS

Dodaj blok Custom HTML w edytorze i wklej kod iframe. Plik game.html wgraj przez FTP do katalogu motywu.

Edytor → Blok HTML → wklej iframe

Przeglądarka → Otwórz plik → game.html
